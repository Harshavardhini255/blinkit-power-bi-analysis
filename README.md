🛒 Blinkit Sales Dashboard | Power BI Project

📊 Project Overview

This project presents an interactive sales dashboard for Blinkit – India's Last Minute App, developed using Microsoft Power BI.

The dashboard analyzes sales performance, product categories, outlet sizes, and customer ratings to generate meaningful business insights.

This project demonstrates skills in data visualization, dashboard design, and business intelligence analysis.

---

📷 Dashboard Preview

"Blinkit Sales Dashboard" (blinkit-sales-dashboard.png)

---

🎯 Business Objectives

The main objectives of this dashboard are:

- Analyze total sales performance
- Identify top-performing product categories
- Compare sales by outlet size and location
- Understand customer ratings and product demand
- Provide data-driven insights for business decisions

📈 Dashboard Insights

1️⃣ Sales by Outlet Establishment

Analyzes how sales vary depending on when outlets were established.

2️⃣ Sales by Item Type

Shows which product categories generate the most revenue, including:

- Fruits & Vegetables
- Snack Foods
- Household Products
- Frozen Foods
- Dairy

3️⃣ Fat Content Analysis

Compares customer preference between:

- Low Fat products
- Regular products

4️⃣ Outlet Size Performance

Sales are analyzed based on outlet size:

- Small
- Medium
- High

5️⃣ Outlet Location Analysis

Sales distribution across outlet tiers:

- Tier 1
- Tier 2
- Tier 3

6️⃣ Outlet Type Performance

Performance comparison among:

- Supermarket Type 1
- Supermarket Type 2
- Supermarket Type 3
- Grocery Stores
